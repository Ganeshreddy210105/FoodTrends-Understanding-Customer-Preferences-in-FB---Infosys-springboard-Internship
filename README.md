# 🍔 Food-Trends: Understanding Customer Preferences on Facebook (FB)

## 📌 Project Overview

**Food-Trends: Understanding Customer Preferences on Facebook (FB)** is a data analytics project focused on analyzing customer food preferences using both structured survey data and unstructured Facebook comments.

The project transforms raw customer feedback into meaningful insights using:

- 📊 Sentiment Analysis  
- 📈 Trend & Regional Analysis  
- 📉 Customer Satisfaction Metrics  
- 📌 Interactive Power BI Dashboards  

The goal is to help businesses understand customer behavior, identify trending food items, and improve decision-making strategies.

## 🔍 Core Functionalities

### 1️⃣ Data Collection & Processing
- Collected survey responses and customer ratings
- Gathered Facebook comments
- Cleaned and standardized datasets
- Removed duplicates and inconsistencies
- Designed unified data model using:
  - `product_id`
  - `customer_id`

---

### 2️⃣ Trend & Regional Analysis
- Identified top-performing food items
- Compared food preferences across regions
- Analyzed category-wise performance
- Generated insights on trending cuisines

---

### 3️⃣ Sentiment Analysis
- Classified Facebook comments into:
  - ✅ Positive
  - ⚖️ Neutral
  - ❌ Negative
- Implemented keyword-based theme extraction
- Compared sentiment across demographics:
  - Age
  - Gender
  - Income

---

### 4️⃣ Customer Satisfaction Measurement
- Calculated average ratings
- Linked ratings with sentiment scores
- Identified key drivers of satisfaction & complaints
